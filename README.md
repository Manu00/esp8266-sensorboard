# esp8266-sensorboard
With this small project i make my first steps into prototyping and pcb-design, using the esp8266.

## Shopping List
- resistor 10k 3x
- resistor 1k 1x
- button 2x
- micro usb female 1x
- ams1117 3v3 1x
- esp12F 1x
- capacitor 100uF 1x
- capacitor 10uF 1x

Optional:
- BME/P 280 1x
